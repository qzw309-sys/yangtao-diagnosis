# 杨涛宇宙 · 创业诊断 yangtao-diagnosis

> 带着你的问题来：查账、找病根、开一个今天就能做的药方。蒸馏自杨涛（涛哥）约 **970 万字**一手输出，自带 1733 条内容单元弹药，装这一个就能独立使用。

## 这是什么

「杨涛宇宙」四件套里的**诊断台**。你说出自己的困境，它用涛哥十九年互联网创业的判断框架帮你查账、点破病根、给出最小行动：

- 「小红书做了 3 个月没流量，怎么办」
- 「客户总嫌贵，我要不要降价」
- 「副业月入八千，要不要辞职」
- 「有人拉我合伙开店，去不去」
- 「实体店快撑不住了，转不转线上」

## 它和普通 AI 回答的区别

- **是涛哥在跟你聊**：全程第一人称对话，像在微信上找到了他本人——「我给你算笔账」
- **先算账**：开口先要数字、先给数字——成本 × 1.5，现金流 > 利润 > 规模
- **反模式预警**：你问怎么做 IP，它先问你业务在哪——IP 是副产物，不是目标
- **有弹药**：回答背后是 1733 条真实内容单元，不是模型的通识想象
- **给动作**：每次收尾都是一个今天就能做的最小动作，不是「加油」
- **不爹味**：像战友，不像导师
